Instructions

Want to learn more about available biking resources in the GTA?

To get a comprehensive overview of cycling resources...

Zoom into the map and watch the clusters change to show you the highest density of bike shops, parkings, and shares at any zoom level.

Note the classification system used to group all GTA bikeways into the 6 categories of Bike Lane, Cycle Track, Multi-Use Trail, Sharrows, Paved Shoulder, and Hiking/Park Trail. See the Definitions tab for more information on this classification.

Have a preference for biking near specific facilities, on specific biking infrastruture, or under specific conditions? Want to know where you should bike?

To filter bike facilities and view current conditions...

Use the toggle menu by clicking on the check boxes for each of the bicycle shops, parkings, and shares to control which of these facilities are visible on the map.

Use the dropdown menu for GTA streets to select whether traffic or bikeways (1 type or all types) are visible on the map. Note that traffic conditions are retrieved from the Mapbox Traffic API - see the Data tab for more information.

Use the weather dropdown menu to select which weather conditions (temperature, precipitation, snow, wind speed) to view on the map. Note that weather conditions are retrieved from the Open-Meteo Weather API and are computed at the centroid of each GTA municipality. Weather conditions are updated every 3 hours. Temperature is specified at 2m above ground, total precipitation is specified as the sum of rain, showers, and snow over the last available hour, snowfall is specified over the last available hour, and wind speed is specified at 10m above ground. See the Data tab for more information.

Have a particular bike facility in mind?

To find out more about a given bike facility...

Use the geocoder search menu to locate any bike shop, parking, share or bikeway on the map.

Click on the desired facility to bring up a popup with available important information about the facility. Note that cursors change when hovering over any element that can be clicked. Also note that the opacity of bikeways changes when hovered to help you locate their start and end points.

For Toronto BikeShare...
Where information is available, popups show the name, address, live number of available bikes and live number of available docks. Note that this live data is retrieved from the Toronto BikeShare API which updates in real-time - see the Data tab for more information.

For Toronto bike shops...
Where information is available and applicable, popups show the name, address, postal code, ward, unit number, city, phone number, email, and whether the shop provides rentals.

For GTA (outside of Toronto) bike shops...
Where information is available and applicable, popups show the name, address, postal code, unit number, city, phone number, email, website, FaceBook site, and open hours.

For Toronto bike parkings...
Where information is available and applicable, popups show the name, address, postal code, ward, city, parking type (rack, stands, etc.), and parking capacity.

For GTA (outside of Toronto) bike parkings...
Where information is available and applicable, popups show the name, whether the parking is covered, whether the parking has a fee, the parking type (rack, stands, etc.), and parking capacity.

For Toronto bikeways...
Where information is available and applicable, popups show the name, the bikeway type on one side of the street as specified by the data source (Facility 1), the bikeway type on the other side of the street as specified by the data source (Facility 2), and how the bikeway was classified into the 6 categories of the map (Classification).

For York Region bikeways...
Where information is available and applicable, popups show the name, the bikeway type as specified by the data source (Facility), how the bikeway was classified into the 6 categories of the map (Classification), the surface that the bikeway is on, and the municipality of York Region that it is located in.

For Peel Region bikeways...
Where information is available and applicable, popups show the name, the bikeway type as specified by the data source (Facility), how the bikeway was classified into the 6 categories of the map (Classification), the surface that the bikeway is on, and the municipality of Peel Region that it is located in.

For Durham Region bikeways...
Where information is available and applicable, popups show the name of the bikeway. The bikeway type was not available, hence the classifications of all such bikeways is listed as 'Not Available'.

For Milton bikeways...
Where information is available and applicable, popups show the bikeway type as specified by the data source (Facility), how the bikeway was classified into the 6 categories of the map (Classification), and the surface that the bikeway is on.

For Burlington bikeways...
Where information is available and applicable, popups show the street name, the bikeway type as specified by the data source (Facility), and how the bikeway was classified into the 6 categories of the map (Classification). Note that the data source for bikeways in Burlington groups them using an acronym system - see the Burlington data source in the Data tab for more information.

For Oakville bikeways...
Where information is available and applicable, popups show the bikeway type as specified by the data source (Facility), how the bikeway was classified into the 6 categories of the map (Classification), and the surface that the bikeway is on. Note that the data source for bikeways in Oakville groups them using a numbering system - see the Oakville data source in the Data tab for more information.

For Ajax bikeways...
Where information is available and applicable, popups show the bikeway type as specified by the data source (Facility), how the bikeway was classified into the 6 categories of the map (Classification), and the street name.

For Whitby bikeways...
Where information is available and applicable, popups show the bikeway type as specified by the data source (Facility), how the bikeway was classified into the 6 categories of the map (Classification), and the street name.

Planning a biking trip?

To find resources for your next biking trip...

Click the "Plan Your Trip!" menu. It will prompt you to select any location on the map by clicking on it - the geocoder search function can also be used to find a location to click on.

Once the location is selected, click "GO". A slider will appear which allows you to select the size of a circular buffer region surrounding the point. Toggle the slider to select desired buffer size.

For every selected buffer, a list of all available amenities (bike shops, bike parkings, bike shares) within the buffer radius and all bikeways that intersect the buffer radius, as well as a summary of weather conditions at the clicked point, will be provided in the planning menu.

Click on any bike shop, parking, or share listed in the planning menu to centre the map around it and bring up its popup with important available information.

To select a new point, click 'CLOSE' and repeat these steps.
